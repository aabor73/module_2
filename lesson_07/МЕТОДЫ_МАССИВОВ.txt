Понимание дополнительных методов массивов:

ИТЕРАТОРЫ

.forEach() используется для выполнения одного и того же кода для каждого элемента в массиве, но не изменяет массив и возвращает undefined.

.map() выполняет один и тот же код для каждого элемента в массиве и возвращает новый массив с обновленными элементами.

.filter() проверяет каждый элемент в массиве, чтобы увидеть, соответствует ли он определенным критериям, и возвращает новый массив с элементами, которые возвращают соответствие критериям.

.findIndex() возвращает индекс первого элемента массива, который удовлетворяет условию в функции обратного вызова. Он возвращает -1, если ни один из элементов в массиве не удовлетворяет условию.

.reduce() выполняет итерацию по массиву, принимает значения элементов и возвращает единственное значение.

Все методы итератора принимают функцию обратного вызова, которая может быть предопределенной функцией, функциональным выражением или функцией со стрелкой.


Инструкции

Определите функцию обратного вызова, прежде чем использовать ее в итераторе.

Объедините два метода итерации в цепочку в одном массиве.

Используйте необязательные аргументы в итераторе для включения индекса или всего массива.

(Ознакомьтесь со страницей Методов итерации массива MDN для получения дополнительной информации)

Используйте .reduce(), чтобы взять многослойный массив и вернуть однослойный массив с нуля.
